三 · 脚本可用 API（作为参考）
Effect.damage(actor, target, 数值, { ignoreDef?: boolean, mul?: 倍率 }) — 造成伤害
Effect.heal(target, 数值) — 治疗
Effect.teleport(actor, x, y) — 瞬移
const target = await Effect.chooseEnemy(actor, { range: {shape:'+', n:4}, hintText:'提示' }) — 选敌人
const cell = await Effect.chooseCell(actor, { range: {shape:'square', n:2}, mustEmpty:true, hintText:'提示' }) — 选格子
Effect.mark(target, '标记key', { display: '显示', modifiers: { atkBuff: 10, defBuff: 5 }, data: {...} }) — 加标记
Effect.unmark(target, '标记key') — 移除标记
Effect.resetAction(actor) — 恢复行动/攻击
Effect.push(actor, target, [dx,dy], 距离) — 击退
Effect.changeTerrain(x, y, 'r'|'m'|'w'|'f'|'plain') — 改变地形
Effect.getEffectiveAttack(actor) — 获取当前攻击力（含加成）
Range.cellsInRange(shape, n, x, y, { includeSelf: true }) — 获取范围内坐标数组
global.Game.pieceAt(x, y) — 查询某格棋子
global.Game.log('文字') — 输出战报
被动技能触发时机 trigger：onKill（击杀后）、turnEnd（回合结束）
