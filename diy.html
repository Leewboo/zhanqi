教程 · 速查
▶ 30 秒上手›
  1. 在左侧「武将属性」填 id、name、hp、atk、def、移动/攻击范围。
  2. 在右侧「技能列表」点「+ 添加」，写 filter 与 content 代码。
  3. 点「✓ 提交」上传到服务器。
  4. 回到游戏首页，DIY 武将会出现在选将池中。
▶ 主动技能示例›

filter：return actor && actor.alive && !Effect.isSkillUsed(actor, skill.id);

const target = await Effect.chooseEnemy(actor, {
  range: { shape: "+", n: 3 },
  hintText: "请选择敌人"
});
if (!target) return false;
Effect.markSkillUsed(actor, skill.id);  // 标记本技能已用（每个技能独立）
Effect.damage(actor, target, Effect.getEffectiveAttack(actor), { mul: 1.5 });
global.Game.log(actor.name + " 发动技能！");
return true;

提示：每个主动技独立计数，武将一回合可释放多个不同主动技。被动技无需调用 markSkillUsed。

▶ 独立技能计数系统›

核心机制：每个主动技本回合可独立释放一次。一个武将拥有多个主动技时，可以分别释放，互不影响。

filter 中检查（推荐）：

return actor && actor.alive && !Effect.isSkillUsed(actor, skill.id);

content 中标记（推荐）：

Effect.markSkillUsed(actor, skill.id);

战报显示：侧栏每个主动技显示「已用/总数」，如「技 1/2」表示本回合已用 1 个、共 2 个主动技。

回合重置：每个回合开始时所有主动技自动重置为「未使用」。

兼容说明：旧式 actor.skilled = true 仍可工作，引擎会自动转换为 Effect.markSkillUsed，仅禁用当前技能而非全部主动技。新代码请直接使用 Effect.markSkillUsed。

▶ 被动技能示例（onKill）›

type：被动 · trigger：onKill

Effect.resetAction(actor);
global.Game.log(actor.name + "【常胜】气势如虹！");
▶ 被动触发时机一览›
  • turnStart — 回合开始时触发（己方回合，可加护盾/buff）
  • turnEnd — 回合结束时触发（己方回合）
  • onKill — 击杀敌人时触发（context.target 为被击杀者）
  • onKilled — 自身被击杀时触发（context.killer 为击杀者）
  • onAttack — 发起攻击时触发（context.target 为攻击目标）
  • onAttacked — 被攻击后触发（context.attacker 为攻击者，context.damage 为受到的伤害）
  • onMove — 移动结束后触发（context.from/context.to 为起止坐标）
  • onHeal — 被治疗时触发（context.healer 为治疗者，context.amount 为治疗量）
  • onSkillCast — 任意技能发动后触发（context.caster 为施法者，context.skill 为技能）

▶ API 速查（战斗类）›
Effect.damage(actor, target, amt, opts) — 造成伤害，opts: ignoreDef/mul
Effect.heal(target, amt) — 恢复生命
Effect.leech(actor, target, amt, opts) — 吸血伤害，leechRatio 默认 0.5
Effect.explode(actor, x, y, n, amt, opts) — 在(x,y)爆炸造成范围伤害
Effect.basicAttack(actor, target) — 普通攻击伤害
Effect.getEffectiveAttack(actor) — 当前攻击力（含加成）
Effect.getEffectiveDefense(target) — 当前防御力（含地形加成）
Effect.modifyAttack(target, delta, turns) — 增减攻击（带持续回合）
▶ API 速查（状态类）›
Effect.shield(target, amt) — 加护盾，先扣盾再扣血
Effect.stun(target, turns) — 眩晕，跳过行动
Effect.charm(actor, target, turns) — 魅惑，目标临时加入己方
Effect.dodge(target, chance) — 获得闪避（0-1概率，下次受击判定）
Effect.thorns(target, amt, turns) — 荆棘反伤护甲
Effect.mark(target, name, opts) — 加标记（display/modifiers/data）
Effect.unmark(target, name) — 移除标记
Effect.hasMark(target, name) — 是否有某标记
Effect.modifyMoveRange(target, delta, turns) — 增减移动力
Effect.markSkillUsed(actor, skill.id) — 标记本技能本回合已用（独立计数）
Effect.isSkillUsed(actor, skill.id) — 查询本技能本回合是否已用
▶ API 速查（位移/选择/工具）›
Effect.chooseEnemy(actor, opts) — 选择敌人，返回目标或 null
Effect.chooseAlly(actor, opts) — 选择友方
Effect.chooseCell(actor, opts) — 选择格子，返回 {x,y} 或 null
Effect.teleport(actor, x, y) — 瞬移到空格
Effect.push(actor, target, dir, n) — 击退 n 格，dir=[dx,dy]
Effect.pull(actor, target, n) — 拉拽 n 格向自己
Effect.resetAction(actor) — 重置移动/攻击/技能
Effect.getAllies(actor) — 获取所有友方存活棋子
Effect.getEnemies(actor) — 获取所有敌方存活棋子
Effect.healArea(actor, shape, n, amt) — 范围治疗友方
Effect.changeTerrain(x, y, type) — 改地形：r(河)/m(山)/f(林)/w(沼)/plain
Effect.random(min, max) — 随机整数
Effect.chance(p) — 概率判定，返回 true/false
global.Game.log(text) — 写战报
global.Game.pieceAt(x, y) — 查询某格棋子
Range.cellsInRange(shape, n, x, y, opts) — 范围格子数组
