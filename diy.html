教程 · 速查
▶ 30 秒上手（点击展开/收起）
  1. 填武将属性（id / name / hp / atk / def / 移动范围 / 攻击范围）
  2. 点「+ 添加一个技能」，写 filter（判断能否施放）和 content（技能主逻辑）
  3. 输入密码（默认 diy123），点「提交到服务器」
  4. 回到首页开一把，DIY 武将会出现在选将池中
▶ 完整示例：技能「强袭」（十字3格 · 1.5倍伤害）
filter(actor) — 返回 true 表示当前可以施放
return actor && actor.alive && !actor.skilled;
content(actor) — 技能主逻辑
// 让玩家选一个敌人
const target = await Effect.chooseEnemy(actor, {
  range: { shape: "+", n: 3 },
  hintText: "【强袭】请选择敌人"
});
if (!target) return false;

// 标记本回合已施放
actor.skilled = true;

// 造成 1.5 倍攻击力伤害
Effect.damage(actor, target, Effect.getEffectiveAttack(actor), { mul: 1.5 });
global.Game.log(actor.name + " 发动【强袭】！");
return true;

▶ 范围形状参数
移动范围 / 攻击范围 / 技能范围均使用 {shape, n} 描述：
  • shape: "+" 十字形（上下左右），n = 距离
  • shape: "square" 正方形，n = 半径
  • shape: "r" 圆形，n = 半径
  • shape: "x" X 形（斜向），n = 距离
