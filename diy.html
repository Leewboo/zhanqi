▶ 30 秒上手›
  1. 在下方「武将属性」填 id（英文）、name（中文）、hp、atk、def、移动范围、攻击范围。
  2. 点「+ 添加一个技能」，每个技能写 filter（是否可施放）和 content（技能主逻辑）。
  3. 在「提交」处输入密码 diy123，点「提交到服务器」。
  4. 回到游戏首页开一把，DIY 武将会出现在选将池中。
▶ 完整示例：技能「强袭」（十字3格 · 1.5倍伤害）›

filter（主动技能，判断是否可施放）：

return actor && actor.alive && !actor.skilled;

content（技能主逻辑，可使用 await）：

// 让玩家选择一个敌人
const target = await Effect.chooseEnemy(actor, {
  range: { shape: "+", n: 3 },
  hintText: "【强袭】请选择敌人"
});
if (!target) return false;

// 标记当前回合已使用技能
actor.skilled = true;

// 造成 1.5 倍攻击力伤害
Effect.damage(actor, target, Effect.getEffectiveAttack(actor), { mul: 1.5 });

// 写战报
global.Game.log(actor.name + " 发动【强袭】！");
return true;

Effect.resetAction(actor);

▶ 范围形状参数›

移动范围 / 攻击范围 / 技能范围均使用 { shape, n } 描述：

  • shape: "+" — 十字形（四向），n = 距离
  • shape: "square" — 正方形（八向），n = 半径
  • shape: "r" — 圆形（曼哈顿），n = 半径
  • shape: "x" — X 形（斜向），n = 距离
▶ API 速查（content / filter 中可直接使用）›
Effect.chooseEnemy(actor, { range, hintText }) — 让玩家选敌人，返回目标或 false
Effect.chooseCell(actor, { range, mustEmpty, hintText }) — 选格子
Effect.damage(actor, target, num, { ignoreDef?, mul? }) — 造成伤害
Effect.heal(target, num) — 回血
Effect.teleport(actor, x, y) — 瞬移
Effect.push(actor, target, [dx,dy], dist) — 击退
Effect.mark(target, 'key', { display, modifiers, data }) — 加标记
Effect.unmark(target, 'key') — 移除标记
Effect.resetAction(actor) — 重置行动
Effect.changeTerrain(x, y, 'r'|'m'|'w'|'f'|'plain') — 改地形
Effect.getEffectiveAttack(actor) — 当前攻击力（含加成）
Range.cellsInRange(shape, n, x, y, { includeSelf }) — 范围格子数组
global.Game.pieceAt(x, y) — 查询某格棋子
global.Game.pieces — 全部棋子数组
global.Game.log(text) — 写战报
actor.x / actor.y / actor.hp / actor.side — 武将属性
target.hp / target.name / target.alive — 目标属性
